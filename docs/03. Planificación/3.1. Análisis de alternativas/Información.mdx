## 1. Resumen ejecutivo

El documento ofrece un análisis de tecnologías clave para el desarrollo y despliegue de aplicaciones. Se abordan herramientas de backend como FastAPI y Django-CRM, y de frontend como React y Next.js. También se evalúan plataformas de alojamiento en la nube como Google Cloud y Vercel. El objetivo es proporcionar información concisa sobre cada posible opción para decidir cuáles serán las escogidas para llevar a cabo el desarrollo de la aplicación.

## 2. Introducción

En este documento, se plantean una variedad de tecnologías utilizadas en el desarrollo de aplicaciones web, abarcando tanto el backend como el frontend, junto con herramientas asociadas y opciones de despliegue, con el objetivo de escoger la mejor combinación de ellas para el desarrollo del proyecto. Desde frameworks hasta herramientas de análisis de código y soluciones de diseño, se explicarán las opciones disponibles para satisfacer las necesidades de desarrollo y despliegue de la aplicación web.
