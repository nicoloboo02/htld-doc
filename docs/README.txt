Todas las carpetas y documentos deben estar numerados. La numeración determina el orden en el que
se muestran en el índice de la página web.

Para que el número de las carpetas no sea visible en la página web, hay que indicar el nombre de
la carpeta en el apartado "label" del archivo "_category_.json" que se encuentra en cada carpeta.

Para que el número de los documentos no sea visible en la página web, hay que indicar el nombre
del documento en la primera línea de cada documento, escribiendo el siguiente texto en formato
Markdown: # <Nombre del documento>
